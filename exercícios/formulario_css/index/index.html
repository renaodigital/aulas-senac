Formulários

Um formulário é usado para coletar informações de entrada. Os dados informados pelo usuário são geralmente enviados a um servidor para o processamento

Exemplo:

 
<form action="#">
    <label for="nome"> Nome:</label> <br>
    <input type="text" id="nome" name="nome"><br>
    <label for="snome">Sobrenome:</label> <br>
    <input type="text" id="snome" name="snome"><br><br>
    <input type="submit" value="Enviar">
</form>




Elementos

Form: usado para criar um formulário em HTML.

Label: é uma espécie de etiqueta. Muito útil para leitores de tela, já que o leitor de tela lerá o rótulo/etiqueta em voz alta. Também ajuda usuários que tem dificuldade em clicar em regiões muito pequenas (por exemplo os botões de radio ou checkbox. O atributo for da tag label deve ser igual ao atributo id do elemento input para poder uni-los.

Input: o input pode ser exibido e várias maneiras, dependendo do atributo type. Por padrão seu type é text

Select: define uma lista suspensa. Acompanhado do elemento <option>, que define uma opção que pode ser selecionada. Por padrão o primeiro item da lista é o selecionado. Para definir uma opção pré-selecionada, usamos o atributo selected. O atributo size serve para especificar a quantidade de valores visíveis na lista e o atributo multiple permite que o usuário selecione mais de um item.

Textarea: define um campo de entrada multilinhas/área de texto. Possui os atributos rows e cols que especificam a quantidade de linhas e colunas visíveis respectivamente.

Fieldset e Legend: Fieldset é usado para agrupar dados relacionados em um formulário e o legend define uma legenda para o fieldset.

Tipos de entrada

O elemento input possui diferentes tipos de entrada:

  • Text: define um campo de entrada de linha única;
  • Password: define um campo de senha;
  • Submit: botão para enviar os dados do formulário;
  • Reset: botão para limpar os valores preenchidos no formulário;
  • Radio: botão de opções que permite que o usuário selecione apenas uma opção. Exemplo: Sexo F ou M;
  • Checkbox: define uma caixa de seleção que permite que o usuário selecione zero ou mais opções de escolha;
  • Date: é usado para campos que contém datas. Podemos usar os atributos min e max para adicionar restrições às datas;
  • Email: usados para campos de entrada que contém email;
  • File: define um campo de seleção de arquivo e um botão de "procurar" para fazer uploads de arquivos;
  • Number: define um campo de entrada numérica;
  • Tel:usado para campos de entrada que devem conter número de telefone.
